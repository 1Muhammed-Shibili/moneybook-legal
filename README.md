# MoneyBook Legal Documentation

This folder contains the legal documentation for the MoneyBook app, designed to be hosted on GitHub Pages.

## Files

- **index.html** - Landing page with links to legal documents
- **privacy-policy.html** - Comprehensive privacy policy (GDPR & CCPA compliant)
- **terms-of-service.html** - Terms of Service agreement
- **styles.css** - Styling for all pages
- **DEPLOYMENT.md** - Step-by-step guide to deploy these pages to GitHub Pages

## Quick Deploy

See [DEPLOYMENT.md](DEPLOYMENT.md) for full instructions.

## Important Actions Required

Before deploying:

1. **Update Contact Email**: Replace `support@moneybook.app` with your actual email in all HTML files
2. **Update Jurisdiction**: In `terms-of-service.html`, replace `[Your Jurisdiction]` with your location
3. **Verify All Information**: Review all legal documents to ensure accuracy

## What's Included

### Privacy Policy ✅
- Firebase services disclosure (Auth, Firestore)
- Google Sign-In data collection
- GDPR compliance (EU users)
- CCPA compliance (California users)  
- Data retention and deletion policy
- Account deletion process
- Contact information

### Terms of Service ✅
- Service description and disclaimers
- Account management and deletion
- Subscription terms (for future use)
- Financial disclaimer
- Liability limitations
- Dispute resolution

## After Deployment

Once deployed to GitHub Pages, you'll have URLs like:

```
https://YOUR_USERNAME.github.io/moneybook-legal/privacy-policy.html
https://YOUR_USERNAME.github.io/moneybook-legal/terms-of-service.html
```

Use these URLs in:
- Google Play Console (Privacy Policy field)
- App settings/about page
- Support communications

## Maintaining These Documents

- Update the "Last Updated" date when making changes
- Notify users of significant changes via app notification
- Keep documents consistent with actual app practices
